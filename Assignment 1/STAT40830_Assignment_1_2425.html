Instructions

Write a slide presentation (15-20 slides) by completing the tasks below on the Economic, Social, Environmental, Health, Education, Development and Energy data on at least three countries. (Below you will find instructions on how to download them).

Use only the R package data.table unless when explicitly stated.

  1. Use data.table to read in the data and assign the correct class to the variables.
  2. Merge the data datasets using data.table.
  3. Do some quick data exploration to know more about your data. (You are allowed to use any R package to answer this question)
  4. Do some analysis on your data using data.table. You can select which variables to use, and which levels of the factors you may be focus on. You must make use of the keyby argument at least once.
  5. Do at least 2 plots using some output from the analysis done in step 3. (You are allowed to use any R package to answer this question).

  • Complete your assignment creating a Quarto presentation, check that all code and output are correctly displayed in your final document. (You may want to adjust the font size for a single slide/whole document, or the printing options for some R output, e.g. see help(print.data.table)).

  • You must add some extra not-default options to personalise the aesthetic of the slides. Upload the .css or the .tex file if use them for the personalisation.

  • Write some prose to explain what you intend to do at each step.

  • Remember that the code should be easy to read and to work with. You should add appropriate comments, make code efficient and use sensible notation.

How to get the data:

For Assignment 1 you will have to download the dataset of the Economic, Social, Environmental, Health, Education, Development and Energy for three different countries of your choice from data.humdata.org/dataset/?q=Economic%2C+Social%2C+Environmental%2C+Health%2C+Education%2C+Development+and+Energy&sort=title_case_insensitive+asc&dataseries_name=World+Bank+-+Economic+Social+Environmental+Health+Education+Development+and+Energy:

  • Click over COUNTRYNAME - Economic, Social, Environmental, Health, Education, Development and Energy and download the dataset called Combined Indicators for COUNTRYNAME it will give you a .csv file called indicators_COUNTRYCODE.csv. (The dataset contains the following variables: Country Name, Country ISO3, Year, Indicator Name, Indicator Code, Value)

  • repeat this procedure for two other countries (total of three countries).

  • notice that some indicators may not be available for all countries.
